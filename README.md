# CampusMate

CampusMate 是校园搭子匹配平台的课程展示型 MVP。项目用于演示“需求提出 → Codex 开发 → 团队 Review → 集成验收”的 AI 协作开发流程，不作为正式线上服务使用。

## 当前能力

- 活动发现、关键词搜索、分类、日期和地点筛选。
- 活动详情、发布活动、申请加入、审批、退出、取消、评论、收藏、举报。
- 模拟登录 / 注册，演示账号为“林小羽”“周同学”“陈同学”，密码均为 `123456`。
- 个人中心支持头像上传、系统头像选择、昵称、学院、年级、个人简介和兴趣标签编辑。
- 个人中心展示参加活动数、发起活动数、完成率、信用分、活动历史、成就徽章和完成评价。
- 设置页支持通知偏好、公开 / 私密开关、申请活动时可见资料、活动类型偏好、时间偏好和退出登录。
- 通知中心支持全部已读、点击进入对应活动、删除通知。
- 本地数据导出 JSON、一键恢复演示数据。
- 简易管理员演示视角：数据概览、活动审核、活动下架 / 恢复、举报列表和处理。
- `localStorage` 本地持久化，所有演示数据仅保存在当前浏览器。
- 全局错误兜底页和 404 页面。
- Playwright 关键流程测试。

## 技术栈

- Next.js 16 + React 19
- TypeScript
- Tailwind CSS
- Playwright
- 浏览器 `localStorage`

## 本地运行

```bash
npm install
npm run dev
```

如果 Windows PowerShell 拦截 `npm`，请使用：

```powershell
npm.cmd install
npm.cmd run dev
```

打开 <http://localhost:3000>。

## 项目边界

这是课堂展示项目。当前版本不接入真实学校账号、云数据库、支付、地图、实时聊天或多设备同步。管理员视角也仅用于本地演示，不代表正式权限系统。

登录页目前展示演示账号和统一密码，但没有一键快捷登录按钮；这样可以保留完整登录流程，课程演示中已经足够。
